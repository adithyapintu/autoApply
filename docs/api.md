# API Design

All endpoints are versioned under `/api/v1`. Authenticated endpoints require `Authorization: Bearer <access_token>`.

## Auth

- `POST /api/v1/auth/register`
- `POST /api/v1/auth/login`
- `POST /api/v1/auth/refresh`
- `POST /api/v1/auth/logout`
- `POST /api/v1/auth/forgot-password`
- `POST /api/v1/auth/reset-password`
- `POST /api/v1/auth/verify-email`
- `GET /api/v1/auth/google/login`
- `GET /api/v1/auth/google/callback`
- `GET /api/v1/users/me`
- `PATCH /api/v1/users/me`

## Resumes and Profiles

- `POST /api/v1/resumes`
- `GET /api/v1/resumes`
- `GET /api/v1/resumes/{resume_id}`
- `POST /api/v1/resumes/{resume_id}/parse`
- `POST /api/v1/profiles/generate`
- `GET /api/v1/profiles/me`
- `PATCH /api/v1/profiles/me`

## Jobs

- `GET /api/v1/jobs/search`
- `GET /api/v1/jobs/{job_id}`
- `POST /api/v1/jobs/{job_id}/match`
- `POST /api/v1/jobs/discovery-runs`
- `GET /api/v1/jobs/discovery-runs/{run_id}`

## Application Materials

- `POST /api/v1/applications`
- `GET /api/v1/applications`
- `GET /api/v1/applications/{application_id}`
- `PATCH /api/v1/applications/{application_id}`
- `POST /api/v1/applications/{application_id}/resume-version`
- `POST /api/v1/applications/{application_id}/cover-letter`
- `POST /api/v1/applications/{application_id}/questions`

## Automation

- `POST /api/v1/automation/tasks`
- `GET /api/v1/automation/tasks/{task_id}`
- `POST /api/v1/automation/tasks/{task_id}/approve`
- `POST /api/v1/automation/tasks/{task_id}/cancel`

## Email, Analytics, Notifications

- `POST /api/v1/email/accounts`
- `POST /api/v1/email/sync`
- `GET /api/v1/email/events`
- `GET /api/v1/analytics/summary`
- `GET /api/v1/analytics/timeline`
- `POST /api/v1/notifications/test`
- `GET /api/v1/notifications`

## Admin

- `GET /api/v1/admin/audit-logs`
- `GET /api/v1/admin/system-settings`
- `PATCH /api/v1/admin/system-settings/{key}`

OpenAPI is generated by FastAPI at `/docs` and `/openapi.json`.
